# PharmacyFX

## About 
This is a simple Pharmacy GUI created using Java Swing. It is the FX version of a Basic Pharmacy GUI I had created using Swing [https://github.com/glomukabwa/Basic-Pharmacy-GUI-Java-Swing]. It implements the CRUD functionalities.

## Features
- Signing up and Logging In<br>
- Adding drug records<br>
- Searching and editting drug records<br>
- Deleting drug records<br>

## Technologies Used
- Java FX<br>
- MySQL<br>

## Future Improvements
- A centralized dashboard to replace the need for different pages for functionalities
- An admin dashboard to manage employees and all drug info
